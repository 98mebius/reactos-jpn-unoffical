1. ReactOSについて
-------------------

ReactOS™は、オープンソースで開発されている、Microsoft®Windows™NT互換のOSです。
Windows用に作成されたアプリケーションおよびドライバーとの互換性の実現を目指しています。

（ここでのWindows™NTとは、NT4, 2000, XP, Vista, 7, 8, 10などのNTファミリーを指します。）

詳細については、https：//www.reactos.orgをご覧ください。


2. ReactOS 日本語版について
-------------------

ReactOS 日本語版は、mebius(@98mebius)が独自にビルドした派生版です。
オリジナルのReactOS チームに公式に認められたバージョンではないことを予めご理解ください。

日本語版の詳細についてはhttp://98mebius.starfree.jp/reactos_jpn/をご覧ください。


3. WINEプロジェクトとの関係
-------------------------------------

ReactOSは、WINEプロジェクトと主に
ユーザーモードDLLなどに関する内容を連携して、
互換性を向上させるための努力をしています。

ほかにも、アプリケーションやテストを実施しています。


4.今後の互換性
-----------------------

ReactOSプロジェクトは、現在はWindows Server 2003に焦点を当てていますが、
将来的にはWindows Vista以降との互換性に重点を置いていきます。



5.他のシステムアプリケーションのサポート
---------------------------------------

Windows NTアーキテクチャでは、ReactOSと同様にサブシステムを使用できます。
サブシステムとは、別のAPIの実行を可能にするものです。
ReactOSからアプリケーションを実行できるようにするオペレーティングシステムとして、
私たちはJava、OS / 2、DOSを検討しています。

将来的には、さらに別のものもサポートするかもしれません。


6. ReactOSは合法ですか？
--------------------

はい。
ReactOSのすべてのコードは、開発者によってゼロから作成されています。
開発者が作成するコードはすべて合法的ですし、
Windowsのシステムを理解する際にも違法なことはしていません。

既存の資料を使用した完全にクリーンなリバースエンジニアリングです。

広範なテスト（数千万）を使用して、書籍とWebの両方から知識を得ています。
また、ブラックボックスエンジニアリング手法をパブリックAPIと
プライベートAPIの両方にしています。
